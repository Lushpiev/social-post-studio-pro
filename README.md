# Telegram AutoPost Mobile 2.2

Готовая база для телефонного приложения автопостинга в Telegram.

## Что уже сделано
- несколько фото в одном посте
- предпросмотр поста
- отложенная публикация и очередь
- история публикаций
- установка на экран телефона как PWA
- подготовка под Android APK через Capacitor
- безопасная схема: токен бота хранится на backend, а не внутри APK

## Важный момент
Чтобы приложение публиковало посты и ставило их в очередь, **backend должен быть запущен**.
Сам APK без backend не сможет безопасно хранить токен Telegram-бота.

## Быстрый запуск на компьютере
```bash
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

Открыть в браузере:
- на компьютере: `http://127.0.0.1:8000`
- с телефона в одной сети: `http://IP_КОМПЬЮТЕРА:8000`

## Настройка Telegram
1. Создай бота через BotFather.
2. Добавь его администратором в канал.
3. Вставь `bot_token`.
4. Укажи `chat_id` в формате `@channel_name` или `-100...`.

## Установка на экран телефона
### Android / Chrome
1. Открой приложение.
2. Нажми **Установить**.
3. Если кнопки нет, в меню Chrome выбери **Установить приложение** или **Добавить на главный экран**.

## Реальный Android APK
Ниже уже всё подготовлено, но для финальной сборки нужен установленный Android Studio + Android SDK.
Это стандартное требование Capacitor.

### 1. Установи Node.js LTS
### 2. Установи зависимости
```bash
npm install
```

### 3. Добавь Android-платформу
```bash
npx cap add android
```

### 4. Синхронизируй проект
```bash
npx cap sync android
```

### 5. Открой Android Studio
```bash
npx cap open android
```

### 6. Собери APK
В Android Studio:
- **Build** → **Build Bundle(s) / APK(s)** → **Build APK(s)**

Готовый APK обычно появится внутри `android/app/build/outputs/apk/debug/`.

## Рекомендованная схема для реальной работы
- backend держать на VPS или домашнем мини-сервере
- телефон использовать как удобный клиент
- PWA использовать сразу
- APK собирать как обёртку для этого интерфейса

## Структура
- `app/main.py` — backend FastAPI
- `webapp/` — мобильный интерфейс PWA
- `uploads/` — загруженные фото
- `data/` — база и настройки
- `capacitor.config.ts` — конфиг для Android-сборки
