# Телефон без Windows: как пользоваться и как получить APK

## Самый простой путь — уже как приложение на телефоне
Это приложение можно использовать как **PWA**: открыть в Chrome на Android и установить на главный экран. PWA устанавливается на устройство через браузер при наличии manifest и installable-конфигурации.

### Как поставить на экран телефона
1. Запусти backend на сервере/VPS.
2. Открой адрес приложения в Chrome на Android.
3. Нажми **Установить** или в меню Chrome выбери **Добавить на главный экран / Установить приложение**.

## Если нужен именно APK, без Windows
В этом проекте добавлен workflow GitHub Actions: `.github/workflows/android-apk.yml`.

### Что делать только с телефона
1. Создай репозиторий на GitHub.
2. Загрузи в него **весь этот проект** через браузер GitHub или мобильное приложение GitHub.
3. Открой вкладку **Actions**.
4. Выбери workflow **Build Android APK**.
5. Нажми **Run workflow**.
6. После завершения открой workflow run и скачай artifact **telegram-autopost-debug-apk**.
7. Распакуй artifact на телефоне и установи `app-debug.apk`.

## Важно
- Workflow запускается вручную через `workflow_dispatch`.
- Артефакты GitHub Actions можно скачивать после выполнения workflow.
- Debug APK обычно устанавливается напрямую на Android, но для установки может понадобиться разрешить установку из неизвестных источников.
- Само приложение для публикации в Telegram по-прежнему требует доступный backend, потому что текущая архитектура хранит настройки, очередь и отправку на серверной части.

## Что уже готово

## Если хочешь полностью без сервера
Тогда нужно делать **другую архитектуру**: прямые вызовы Telegram Bot API из самого приложения и урезать/переделывать очередь. Это можно сделать отдельной версией, но текущий проект рассчитан на backend.
